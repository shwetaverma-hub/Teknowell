The Bascis

  •   HTML stands for Hyper Text Mark-up Language, and is the basic code behind web pages.
  •   In Internet Explorer load your favourite web pages. Right click somewhere on the web page, select view source, you will now
  •   You will be using software called Notepad++ to write your HTML code.
  •   An HTML file is just a page of text with lines of code set in tags. The tags are set in angled brackets
  •   The tags tell the browser how to display the contents on the page.
  •   Most tags come in pairs i.e. an Opening and a Closing tag.

   for example

Opening Closing
< HTML > < /HTML >
< TITLE > < /TITLE >
< HEAD > < /HEAD >
< BODY > < /BODY >

  •    One tag that does not need closing tag is the < BR >tag, which is used to force or break the text onto next line. It's the HTML equivalent or hitting the Enter key on your keyboard when you type words in Microsoft Word.
  •    If you like more help with the tags then you can go to
    https://www.w3schools.com/html/ for details of all tags and thier attributes.
